# 🚔 Face Trace - Criminal Sketch AI System

## 🌟 Overview

Criminal Sketch AI System is an intelligent cross-platform application built with Flutter, Node.js, MongoDB, and AI-powered sketch generation technologies. The system allows users to provide criminal descriptions using:

* Text prompts
* Voice input in English
* Voice input in Urdu
* Custom appearance selections

Based on the provided information, the application generates a visual criminal sketch automatically using AI.

The project is designed to assist investigation workflows, reporting systems, and smart criminal identification processes.

# 📱 Frontend Setup (Flutter)

## Navigate to Flutter Project

```bash
cd frontend
```

## Install Dependencies

```bash
flutter pub get
```

## Run Application

```bash
flutter run
```

---

# ⚙️ Backend Setup (Node.js)

## Navigate to Backend Folder

```bash
cd backend
```

## Install Dependencies

```bash
npm install
```

## Start Server

```bash
npm start
```

# 🔑 Environment Variables

Create a `.env` file in backend directory.

```env

PORT=5000
OPENAI_API_KEY=your_api_key
```

# 🔄 Workflow

1. User enters text or voice description
2. Voice converts to text
3. Selected criminal attributes are added
4. Prompt is processed by AI engine
5. Criminal sketch is generated
6. Result is displayed to the user
